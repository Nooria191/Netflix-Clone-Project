## Overview
This is a static Netflix clone built using HTML and CSS. It replicates the basic layout of the Netflix homepage, focusing on design, structure, and responsiveness. The project highlights skills in HTML structuring and CSS styling without using JavaScript.

## Features
Clean and structured layout using HTML
Styled user interface with CSS
Navigation bar similar to Netflix
Hero/banner section design
Content sections with images/thumbnails
Responsive design for different screen sizes
Use of Flexbox and Grid for layout
Hover effects for better user interaction
